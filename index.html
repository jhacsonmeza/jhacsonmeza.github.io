Jhacson Meza

I am an ELLIS PhD student at the Czech Technical University in Prague supervised by Torsten Sattler and Martin Oswald. My research interests lie in 3D vision and 3D reconstruction.

I recevied my MSc degree in Electrical and Electronics Engineering, where I developed a 3D multimodal imaging system by combining the 3D freehand ultrasound and structured light techinques. I received my BSc degree in Mechatronics Engineering working on terrain 3D reconstruction for civil engineering applications using the Structure-from-Motion pipeline.

Open Source Code
  • 3D freehand ultrasound calibration: Python implementation for calibration of a 3D freehand ultrasound system using a stereo vision system and a planar marker.
  • C++ acquisition system of two stereo Basler cameras and an ultrasound machine.
  • Notes about trifocal tensor estimation and point transfer using a calibrated three-view system. A camera1-projector-camera2 setup is used as a three-view system. The trifocal tensor is estimated for that calibrated case and point transfer is applied to the third view (camera2).
  • Notes about fundamental matrix estimation with calibrated stereo cameras, using different approaches: epipolar restrictions and bilinear relations.
  • Notes about stereo triangulation where different methods are evaluated: DLT, inhomogenenous, optimal method, and plane-line triangulation (using a single coordinate for a view and both for the another view).
  • C++ code example for NGIMU data acquisition over Wi-Fi through OSC messages using ospack library.
