Jhacson Meza

I am a Master's student in Engineering and a research assistant at UTB in Colombia with a major in Mechatronic Engineering. I am currently working on developing a low-cost 3D multimodal imaging system for internal and external 3D reconstruction by combining elements from 3D ultrasound imaging, 3D optical sensing, image processing, and deep learning.

I am also interested in using Deep Learning for computer vision applications like keypoint detection for 6D pose estimation or image segmentation. I have experience working on 3D reconstruction using digital fringe projection for biomedical applications. Previously, I worked using the Structure from Motion technique for terrain reconstruction and analysis.

My research interests include Computer Vision, 3D Imaging, Deep Learning, Machine Learning, Image Processing, Medical Imaging, and Optical Metrology.

Open Source Code
  • MarkerPose: PyTorch and LibTorch implementation of the paper "MarkerPose: Robust Real-time Planar Target Tracking for Accurate Stereo Pose Estimation."
  • SL+3DUS: Python implementation of the paper "Three-dimensional multimodal medical imaging system based on freehand ultrasound and structured light."
  • 3D freehand ultrasound calibration: Python implementation for calibration of a 3D freehand ultrasound system using a stereo vision system and a planar marker.
  • C++ acquisition system of two stereo Basler cameras and an ultrasound machine.
  • Notes about trifocal tensor estimation and point transfer using a calibrated three-view system. A camera1-projector-camera2 setup is used as a three-view system. The trifocal tensor is estimated for that calibrated case and point transfer is applied to the third view (camera2).
  • Notes about fundamental matrix estimation with calibrated stereo cameras, using different approaches: epipolar restrictions and bilinear relations.
  • Notes about stereo triangulation where different methods are evaluated: DLT, inhomogenenous, optimal method, and plane-line triangulation (using a single coordinate for a view and both for the another view).
  • C++ code example for NGIMU data acquisition over Wi-Fi through OSC messages using ospack library.
